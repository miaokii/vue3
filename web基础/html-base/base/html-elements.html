基本元素

块级元素和行内元素

块级元素：独占一行的元素

div、h1、h2、marquee、p等

行内元素：只占自身的宽度

input span等

规则1：块级元素中可以写行内元素，块级元素，几乎都能写

规则2：行内元素中可以写只可以写行内元素，不能块级元素

特殊规则：h1-h6不能嵌套，p标签中不能写块元素

文本标签

文本标签基本都是行内元素

em：着重阅读的内容，默认斜体表示：安装国家反诈App，预防诈骗

strong：强调内容，十分重要的内容：安装国家反诈App，预防诈骗

span：没有语义，包裹短语的通用容器：安装国家反诈App，预防诈骗

div：没有语义的块元素，当找不到更好的块级元素、或者不想增加特定的意义时使用

不常用文本标签

cite：标记作品名：《三体》、《平凡的世界》

dfn：特殊术语，专属名词：年轻人你耗子尾汁

del与ins：删除的文本与插入的文本：原价199 ，秒杀99

sup与sub：上标与下标：水的化学方程式：H2O，8的立方：83

code：一段代码：alert(1)

samp：从正常的上下文中，将某些内容提取出来：支付宝到账100万元

kbd：键盘文本，表示文本通过键盘输入，用于计算机相关手册：保存的快捷键：command+s

abbr：缩写，配合title属性：LOL 很好玩

bdo：更改文本方向，配合dir属性，ltr、rtl：你是年少的欢喜 反过来是：你是年少的欢喜

var：标记变量，与code标签一起：定义一个变量let a=1

small：附属细则，版权、法律文本等：

b：关键字：买个 GPW 鼠标，好用

i：呈现字体图标，现用于 表示图标、字符等：燕子，没你我怎么活啊

u：与正常文本有反差的内容，错单词等：Helllo world

q：短引用：其实路上本没有路

blockquote：长引用，块元素：

后来，我总算学会了如何去爱，可你早已消失在茫茫人海

address：地址信息，块元素：

成都市双流区双流机场
