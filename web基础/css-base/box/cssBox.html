元素的显示模式

块元素 block

又称块级元素，特点

  1. 在页面中独占一行，不与任何元素公用一行，从上到下排列
  2. 默认宽度：撑满父元素
  3. 默认高度：由子元素内容撑开
  4. 可以通过css设置宽高
独占一行
css设置宽高
内容撑开高度，仔细看哦

行内元素 inline

内联元素，特点

  1. 不独占一行
  2. 默认宽度：内容撑开
  3. 默认高度：内容撑开
  4. 无法通过css设置宽高
不独占一行 默认宽度：内容撑开，默认高度：内容撑开 无法通过css设置宽高

行内块元素 inline-block

內联块元素，特点

  1. 不独占一行，一行显示不下，继续下一行
  2. 默认宽度：内容撑开
  3. 默认高度：内容撑开
  4. 可以通过css设置宽高

盒模型组成

盒子的大小=content+左右padding+左右border

盒子内容区

设置盒子内容区宽度，有width，min-width，max-width等属性；相应的高度也是

content

内容区宽度，当未设置宽度时，内容区宽度为视口宽度减去margin左右减去border左右减去padding左右

盒子内边距padding

左右那边距可以单独设置

设置上下margin，高度不能被撑开

观察到分割线被遮挡，所以对于行内元素，设置左右内边距，元素的宽度会被撑开，设置上下边距，元素高度不会被撑开

盒子边框border

边框的上下左右颜色，宽度，边框线风格都可以分别设置

也可以只设置一个border复合属性

盒子外边距

外边距可以分别设置上下左右的宽度，也可以使用复合属性

  • 子元素的margin是参考父元素的content内容去计算的
  • 上左margin，影响自己的位置，右下margin影响后面兄弟元素的位置
  • 行内元素设左右margin可以设置，上下margin设置无效
  • margin值可以是auto，给一个块级元素设置左右margin为auto实现其在父元素中水平居中效果
  • margin的值可以是负值

margin塌陷

给第一个子元素添加margin-top和最后一个子元素添加margin-bottom会作用在父元素上

解决：给父元素添加overflow：hidden属性

查看 h3c类选择器样式代码

margin合并

上下相邻的子元素相接的margin设置会合并，并取较大值

布局技巧

行内元素和行内块元素，布局时可以当作父元素的文字处理

子元素在父元素中水平居中

  • 子元素为块元素：给子元素加上margin: 0 auto
  • 子元素为行内元素、行内块元素：父元素加上text-align：center

子元素在父元素中垂直居中

  • 子元素为块元素：设置子元素的margin-top
  • 子元素为行内元素、行内块元素：父元素line-height=height ，每个子元素加上v-align：middle
    若绝对垂直居中，设置父元素font-size：0 ，再分别设置子元素的font-size
