浮动布局

元素浮动后的特点

  • 脱离文档流，元素的宽高默认被内容撑开
  • 不会独占一行，也可以与其他元素共一行
  • 不会margin合并，不会margin塌陷，完美设置padding和margin
  • 不会像行内块一样当作文本处理（没有行内空白的问题）

元素浮动的影响

元素浮动后，会对父元素和兄弟元素产生影戏哪个

  • 对兄弟元素：后面的兄弟元素，会占据浮动元素之前的位置
  • 对父元素：不能撑起父元素的高度，导致父元素高度塌陷，但父元素宽度仍然束缚浮动元素

解决元素浮动影响

  1. 方案1：给父元素设置高度
  2. 方案2：给父元素设置浮动，带来其他影响
  3. 方案3：给父元素设置overflow：hidden
  4. 方案4：给左右浮动元素后面，添加一个块元素，给该元素设置clear：both（清除前面兄弟元素左右浮动的影响）
  5. 方案5：给父元素设置伪元素，通过伪元素清楚浮动，原理同方案四
    .parent::after { content: ''; display: block; clear: both }
浮动原则：设置浮动时，兄弟元素要么全浮动，要么全不浮动
