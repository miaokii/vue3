布局概览

布局类型

  • 正常布局流 none
  • 弹性盒子 flexbox
  • 网格 grid
  • 浮动 inline-block
  • 定位
  • 表格 table
  • 多列

正常布局流

正常布局流表示不对页面进行布局控制，如下布局属性会覆盖默认的布局行为

  • display属性：元素在正常布局流中的表现形式。block、inline...
  • 浮动：设置float值，如left让块级元素互相并排成一行
  • position属性：准确设置盒子的位置
  • 表格布局：在非表格元素上，使用表格属性
  • 多列元素：Multi-column layout让块按列布局等

弹性盒子Flexbox

用于创建横向或纵向的一维页面布局，使用时，在想进行flex布局的父元素上修改display属性为flex

浮动

将元素浮动，会将元素从正常流中移除，浮动到左侧或右侧，其他元素会在浮动元素周围环绕

float属性的四个值：

  • left：浮动到左侧
  • right：浮动到右侧
  • none：默认值，不浮动
  • inherit：继承父元素的浮动属性

定位

将一个元素从正常布局流移动到另一个位置，定位类型有五种：

  • 静态定位static：默认
  • 相对定位relative：相对于元素在正常文档流中的位置移动
  • 绝对定位positioning：将元素从文档流中移出，类似放到单独的图层中，例如通过标签显示和隐藏内容面板或通过按钮控制元素滑动
  • 固定定位fixed：将元素在浏览器视口中固定，类似在整个页面滚动过程中，元素总处于屏幕的某个位置的导航栏菜单
  • 粘性定位sticky：保持静态定位，当相对视口达到某个预设值时，像固定定位一样

相对定位

将元素按照原来的位置偏移

绝对定位

将元素移除布局流，以坐标的形式相对于其容器定位到web页面的任何地方。经常用与浮动和相对定位

    我是一个普通元素
    我是一个绝对定位元素
    我是一个普通元素

固定定位

和绝对定位一样，不同之处在于元素的位置不是基于容器的边框计算，而是视口的边框计算。可用于固定位置的菜单，不受页面滚动的影响

    我是一个普通元素
    我是一个固定定位元素
    我是一个普通元素

粘性定位

静态定位和固定定位的混合，当到达预设定的位置时，不再滚动

    我是一个普通元素
    我是一个粘性定位元素，当我滑动到屏幕顶部500px时，就停止滚动
    我是一个普通元素
